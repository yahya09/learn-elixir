Let's Build with Elixir and Phoenix

Learn to build professional web applications with Elixir and Phoenix Framework

Welcome!

This comprehensive guide teaches you how to build production-ready web applications using Elixir and Phoenix Framework. Perfect for senior engineers transitioning from object-oriented languages like Python, PHP, C#, or JavaScript.

What You'll Build

Throughout this guide, you'll build SnippetBox - a complete web application for creating and sharing text snippets (similar to Pastebin or GitHub Gists).

What You'll Learn

  • Elixir Fundamentals - Pattern matching, immutability, pipe operator, functional programming
  • Phoenix Framework - Controllers, views, templates, contexts, request lifecycle
  • Ecto Database - Schemas, migrations, queries, changesets, transactions
  • Plugs - Middleware pattern for composable request processing
  • LiveView - Real-time, interactive interfaces without JavaScript
  • Authentication - User registration, login, sessions, authorization
  • Testing - Unit tests, integration tests, test-driven development
  • Deployment - Releases, production configuration, Docker

Who This Is For

This guide is designed for senior software engineers who:

  • Have professional experience with web development
  • Are familiar with Python, PHP, C#, JavaScript, or similar languages
  • Have minimal or no experience with functional programming
  • Want practical, hands-on learning rather than pure theory

About This Guide

This guide is modeled after the excellent "Let's Go" book for Golang by Alex Edwards, but adapted for Elixir's functional programming paradigm and Phoenix's conventions. It's designed to help experienced developers make the transition from object-oriented to functional programming while building real-world applications.

Prerequisites: Elixir 1.16+, Phoenix 1.7+, PostgreSQL 14+
Source Code: Available in the guide repository
Community: Elixir Forum | Elixir Slack
