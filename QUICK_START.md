### Essential Commands

```bash
# Create new Phoenix project
mix phx.new myapp

# Get dependencies
mix deps.get

# Create database
mix ecto.create

# Run migrations
mix ecto.migrate

# Start server
mix phx.server

# Run tests
mix test

```
